Computer & OS #

I use my own-built computer along side with an Ideapad gaming laptop. Currently OpenBSD , before writing this post I used to have a windows OS dual-booted in my machine for work purpose1, today alhamdulilah I’ve quit this shit so I deleted Windows permanently, and inshallah I will not download it forevermore.

There is lots of reasons for me to prefer BSDs over Linux. Being a minimalist, I see that the BSD implementation of the UNIX utilities is optimal, readable, less bloated, compatible, and works harmonically in a captivating way with the monolithic kernel. I never had a compatibility issue using a BSD. Unlike Linux, I had to recompile my kernel twice to get Bluetooth to work fine. Also BSD docs rocks. Without being minimalist, the real reason not to get back to Linux is that it would be arduous for me to migrate.

Software #

Window Manager #

I run X11 with suckless’ dynamic window manager, which is an optimal window manager written in C so it is easy for me to read, patch and maintain. It’s less than 500KB, so it’s immovable & efficient as hell. I use many patches with it but I don’t recall names.

File Manager #

I use LF, a terminal-based file manager written in go. In the past days I used to use a GUI file manager, mainly Xfce’s Thunar and the idea of using a terminal file manager looked like an antiquated thing to do. Then I had an issue with dbus,the message bus system, I had to wait ~4 minutes to get any gtk-based program to run, includes thunar. Thus, I had to use lf, it was arduous at the beginning, but after while I automated my file management to the maximum level, and now I can’t bear using a GUI file manager. Here is some pics;

dmenu #

I don’t use suckless’ dmenu, instead I use rofi, which is written in C as well. The nice thing about rofi is that I don’t have to do a lot with it to get it work. And it looks nicer than dmenu. However, it’s a lot bloated if compared to dmenu.

Web Browser #

I hate all web browser, yet I’ve never found a reliable web browser, never. my current web browser is Brave, a bloated web browser, but like they say, to browse a bloated web, you need a bloated web browser. I even bloat it more, to satisfy my needs, with the following web packages;

  • Feebro, a social media & RSS feeds reader. I use it to browse the internet feeds instead of opening the network website services, which mostly sucks and not even retrieve the posts I’m waiting for. Feedbro once sees a new post, it saves it locally on my machine. and I can read it whenever I want.

  • FoxyProxy I mentioned my usage for it before here, in Arabic.

  • Linux Scroll Speed Fix Unix like scroll functionality sucks.

  • Modern scrollbar I don’t like the way scrollbars look like.

Terminal Emulator #

Kitty is my terminal. I used to use suckless’ ST, but I noticed that kitty is much faster. Of course, compared to ST, it’s bloated.

Status Bar #

I used to use the official suckless’ status bar, but I found one written in rust & much easier to configure and git it work. rsblocks.

Media Player, Image Viewer #

MPV is a great video player, I think that MPV is the 2nd best written software in my machine. I patched it with subliminal so it downloads subtitles for movies automatically when I play movie with it. Also I use it as an image viewer.

Music Player #

musikcube “is a fully functional terminal-based music player, library, and streaming audio server that runs natively on Windows, macOS, and Linux. it also runs well on a Raspberry Pi with a custom DAC (e.g. IQaudIO DAC+, HiFiBerry DAC+ and others), and can output 24bit/192k audio comfortably.”
